


<h1>Name</h1>

Weather App

<h1>Description</h1>

Commandline App for getting temperature in any city you like.
App is using OpenMeteo API https://open-meteo.com/

<h1>Prerequisites</h1>
To use this app you need JRE/JDK 17

<h1>Installation</h1>

To run this App you need to download the source and add jar artifact to the project. After building the project you can use the jar app.

<h1>Usage</h1>

                1 - Get the temperature for hardcoded city which is Gdansk
                2 - Get the temperature for whatever city you like
                3 - Start collecting temperature for hardcoded city and calculate trend
                4 - Get historical data for hardcoded city and calculate trend
                5 - Exit
